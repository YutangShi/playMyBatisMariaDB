@* 這邊我們可以發現到兩件事 1.如果要import物件時，寫法跟Java一樣 當有寫好公用工具時，也可以這樣去呼叫 2.宣告變數時，一定要寫在同一行，而不能寫分段寫 var 宣告，意思變數是可以被修改的 3.如果要在頁面上使用變數或是判斷式等等 前面要使用小老鼠符號，才可以順利使用 *@ @import java.math.BigInteger; var java_v=0 @import java.lang.String; var name = "Tom" Java 變數 : @java_v
Java 變數 + 1 : @(java_v+1)
Java 字串 : @name
Java 字串加長 : @(name + " Clancy 湯姆克蘭西：全境封鎖")
